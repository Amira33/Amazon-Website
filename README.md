* Install and Run the Project:

* JDK17 installed
* Maven project 
* Pom.Xml file which has all depenciecies 
* webdrivermanager
* (chrome)
* Framework : TestNG open source with annotation (@Test , @BeforeMethod , @afterMethod) 
* Tstng.Xml : file which have 3 classes of the project to run them and has the <listeners>
        <listener class-name="listeners" which used to print the test case fail or pass in the consolo. 
        
              <class name="com.AmazonApp.testcases.LoginPageTest">
        
            <class name="com.AmazonApp.testcases.TodayDealsTest">

            <class name="com.AmazonApp.testcases.AccountLists">
        

* Framework Used : Page Object Model
